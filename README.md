### 📚 What I learned

Durante esse projeto, consegui evoluir bastante em:

* Estruturação de layouts responsivos com **mobile-first**
* Uso combinado de **Bootstrap + CSS personalizado**
* Organização de código com **boas práticas de CSS**
* Criação de componentes reutilizáveis (cards, navbar, etc.)

Exemplo de estrutura que gostei bastante:

```css
:root {
  --primary-color: #fad400;
  --secondary-color: #fe7867;
}
```

---

### 🚀 Continued development

Nos próximos projetos, quero focar em:

* Melhorar ainda mais minha responsividade (principalmente em telas grandes)
* Aprimorar acessibilidade (a11y)
* Evoluir no uso de JavaScript para interações mais dinâmicas
* Trabalhar com frameworks como React no futuro
